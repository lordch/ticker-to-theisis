# Budimex Periodic Reports Portal

## Page Overview
This is the investor relations page for **Budimex SA**, a Polish construction company, displaying periodic financial reports spanning from 2005-2025.

## Available Report Types

The page offers two main report categories:

1. **Skonsolidowane (Consolidated Reports)** - Group-level financials
2. **Jednostkowe (Individual Reports)** - Budimex SA entity-specific financials

## Report Periods Available

For each year, reports are released in four time intervals:
- **1Q** (January-March)
- **H1** (January-June)
- **3Q** (January-September)
- **FY** (Full Year: January-December)

## Most Recent Reports (2025)

- Q1 2025: PDF and ZIP formats available
- H1 2025: ZIP format available
- Q3 2025: PDF and ZIP formats available

## Key Financial Data (2023)

The page displays these metrics:

## Download Formats

Reports are available as:
